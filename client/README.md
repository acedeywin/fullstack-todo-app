## Getting Started

First, run the development server:

```bash
npm run dev
```

**Production Mode:**

```bash
npm run start
```

### Docker:

**Build the image:**
```bash
npm run docker:build
```
**Run the container:**
```bash
npm run docker:run
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
